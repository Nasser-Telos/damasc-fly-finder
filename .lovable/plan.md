

# خطة تحسين التصميم - إضافة قسم العروض بأسلوب Google Flights

## الهدف

إضافة قسم ترويجي أنيق مشابه لقسم "Flexible? Discover the best flight deals with AI" في Google Flights، لكن بعنوان "أرخص الرحلات لهذا الشهر" - مع الحفاظ على البساطة والوضوح.

## التصميم المرجعي من Google Flights

```text

│  ✨                                                              │

│     Describe your ideal trip, and let Google Flights            │
│     find the best deals for you                                 │
│                                              [Explore deals →]  │

```

```text

│  📅                                                              │

│     شاهد أفضل الأسعار للرحلات إلى دمشق خلال يناير               │
│                                              [عرض التقويم ↓]    │
└─────────────────────────────────────────────────────────────────┘
```

## التغييرات المطلوبة

### 1. إضافة قسم العروض الترويجي (Deals Banner)

**الموقع:** بين بطاقة البحث الرئيسية وقائمة الرحلات

**التصميم:**
- خلفية خفيفة بتدرج أزرق ناعم (مثل Google Flights)
- أيقونة تقويم أو صفقات على اليمين
- عنوان رئيسي واضح
- وصف قصير بسيط
- زر للانتقال إلى قسم التقويم

**الألوان:**
- خلفية: `bg-blue-50` أو تدرج خفيف
- النص: الأزرق الأساسي للعناوين
- حدود: `border-blue-100`

### 2. تحسين قسم التقويم الحالي

**التعديلات:**
- تبسيط العرض أكثر
- إضافة انتقال سلس عند النقر على زر "عرض التقويم"
- تحسين ألوان الأيام حسب السعر

### 3. تحسينات عامة على الألوان والترتيب

| العنصر | الحالي | المقترح |
|--------|--------|---------|
| قسم العروض | غير موجود | بانر أزرق فاتح مع أيقونة |
| التقويم | أسفل الصفحة | يظل في مكانه مع ربط بالبانر |
| الألوان | جيدة | تدرج أزرق أخف للعروض |

## التفاصيل التقنية

### مكون DealsBanner الجديد

```typescript
function DealsBanner({ 
  userCity, 
  cheapestPrice,
  onScrollToCalendar 
}: { 
  userCity: string;
  cheapestPrice?: number;
  onScrollToCalendar: () => void;
}) {
  return (
    <Card className="bg-gradient-to-l from-blue-50 to-white border-blue-100 mb-6">
      <CardContent className="p-4 flex items-center gap-4">
        {/* أيقونة */}
        <div className="w-10 h-10 rounded-full bg-blue-100 flex items-center justify-center">
          <Calendar className="h-5 w-5 text-primary" />
        </div>
        
        {/* النص */}
        <div className="flex-1">
          <h4 className="font-medium text-foreground">
            أرخص الرحلات هذا الشهر
          </h4>
          <p className="text-sm text-muted-foreground">
            شاهد أفضل الأسعار للرحلات من {userCity} إلى دمشق
          </p>
        </div>
        
        {/* الزر */}
        <Button 
          variant="outline" 
          className="text-primary border-primary/30"
          onClick={onScrollToCalendar}
        >
          عرض التقويم
        </Button>
      </CardContent>
    </Card>
  );
}
```

### ربط البانر بالتقويم

سيتم إضافة `ref` لقسم التقويم واستخدام `scrollIntoView` للانتقال السلس:

```typescript
const calendarRef = useRef<HTMLElement>(null);

const scrollToCalendar = () => {
  calendarRef.current?.scrollIntoView({ behavior: 'smooth' });
};
```

## ملاحظات

- التصميم سيبقى بسيطاً جداً ومناسباً لجميع الأعمار
- الألوان ستتبع نظام Google Flights (أزرق + أبيض + رمادي فاتح)
