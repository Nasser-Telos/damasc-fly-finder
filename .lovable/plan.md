

# تحسينات صفحة نتائج البحث - النسخة النهائية

## التحسينات المعتمدة فقط

### 1. شارة "ارخص سعر"
- شارة خضراء صغيرة تظهر فقط على اول بطاقة عند الترتيب بالسعر
- تصميم: خلفية خضراء فاتحة + نص اخضر غامق + رمز نجمة
- موقعها: اعلى البطاقة كعنصر صغير انيق

### 2. تحسين زر الحجز
- تاثير hover: تكبير خفيف (scale 1.05) + ظل ازرق ناعم
- تاثير ضغط (active): تصغير خفيف (scale 0.97)
- اضافة نص "الموقع الرسمي" بخط صغير تحت الزر

### 3. تحسين تصميم بطاقات الرحلات
- جعل البطاقات ابسط واجمل مع مساحات متوازنة
- تحسين الخط بين نقطتي المغادرة والوصول (خط متقطع + نقاط بداية ونهاية)
- تحسين تباين الالوان وتوزيع العناصر داخل البطاقة
- ظل خفيف على البطاقات يزداد عند hover

## التفاصيل التقنية

### تعديل `src/pages/Search.tsx`

**تغييرات على `SearchFlightCard`:**
- اضافة prop `isCheapest` لعرض الشارة الخضراء
- تحسين هيكل البطاقة: شارة ارخص سعر في الاعلى اذا كانت الارخص
- اضافة نص "الموقع الرسمي" تحت زر الحجز
- تمرير `isCheapest` من المكون الاب بناء على `sortBy === "price" && index === 0`

**تغييرات على `SearchPage`:**
- تمرير `index` و `isCheapest` لكل بطاقة في الـ map

### تعديل `src/pages/Search.css`

**شارة ارخص سعر:**
- `.search-fc-badge`: خلفية خضراء فاتحة، نص اخضر غامق، border-radius دائري، حجم خط 11px، مع ايقونة نجمة

**تحسين زر الحجز:**
- `.search-fc-book`: اضافة `transition: all 0.2s`
- `.search-fc-book:hover`: اضافة `transform: scale(1.05)` + `box-shadow: 0 4px 12px hsl(217 91% 60% / 0.25)`
- `.search-fc-book:active`: اضافة `transform: scale(0.97)`
- `.search-fc-book-sub`: نص "الموقع الرسمي" بحجم 10px ولون رمادي تحت الزر

**تحسين البطاقات:**
- `.search-flight-card`: اضافة `box-shadow: 0 1px 3px hsl(220 13% 91% / 0.5)` كظل اساسي خفيف
- `.search-flight-card:hover`: تحسين الظل ليصبح اقوى + تحريك خفيف للاعلى `transform: translateY(-2px)`
- `.search-fc-line-bar`: تحويل من خط مستمر الى متقطع (`border-top: 2px dashed`) مع نقاط في البداية والنهاية
- اضافة `.search-fc-dot`: نقطة دائرية صغيرة (6px) عند بداية ونهاية خط المسار
- تحسين المسافات الداخلية (padding) لتوزيع افضل
